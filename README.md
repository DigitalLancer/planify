# Planify – Personal Event Planner

Planify is a full-stack personal event management application that allows users to create, track, and manage their events in a clean and intuitive interface.

It combines a modern Next.js frontend with a secure ASP.NET Core Web API backend, featuring authentication, user-specific data, and a responsive dashboard.

---

## Features

- **Event Management**
  - Create, update, delete events
  - Categorize events (exercise, entertainment, etc.)
  - Track event status (upcoming, completed, cancelled)
  
- **Authentication & Authorization**
  - Secure login/register system using ASP.NET Identity
  - User-specific event data

- **Weather Integration**
  - Daily weather forecast (Open-Meteo API)
  - Visual weather indicators

- **Modern Frontend**
  - Built with Next.js (App Router)
  - React Query for server state management
  - TailwindCSS for styling
  - Component-based architecture
